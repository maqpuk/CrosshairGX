CrosshairGX — это уникальное приложение для устройств под управлением Windows (10,11), 
которое облегчает жизнь многим любителям многопользовательских игр на ПК. 
Оно позволяет добавить на ваш экран иконку прицела, которая не исчезнет даже после запуска стороннего приложения. 
Такой механизм позволяет точно наводить камеру на цель в тех случаях, когда это не предусмотрено разработчиками игры и получить 
некоторое преимущество перед другими игроками. Важной особенностью приложения является то, 
что оно не является читом и не может отслеживаться администраторами игровых серверов.
